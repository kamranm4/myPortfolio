FactAR

A phone app that uses Augmented Reality to help educate users about their surroundings. Created as a submission for the AI Societal Impact Challenge 2020. This neat app would allow users to tap on anything in their surroundings, and the app would provide fun facts about the object they tapped on. It even includes features to teach the application what facts users like and dislike, and provides opportunities to dive deeper and learn more interesting things about the world around them!

Tools Used: Java, Kotlin, Microsoft Azure Cloud APIs, Figma, Android Studio
